## Member

Member是一个会员管理系统，为企业在自己的网站上吸引、管理自己的客户资源以及注册会员而开发，以此使企业与自己的用户建立良好的沟通与互动。

-----
### 主要技术
  
JSP，Servlet，Java Bean，JDBC
界面：JQuery ,EasyUI
数据库：MySQL  

-----
### 功能  
前台：完成登录，注册，查看个人信息、公告、交易明细，进行点卡冲值、下载资源、添加评论及相册等功能。
后台：查看会员信息以及对不同会员的分类管理。上传供下载的资源、发布公告、点卡生成等功能。
